# **App Name**: CrioCaseira Quizz

## Core Features:

- Quizz questions: Displays the series of questions related to weight loss and personal habits
- Weight and measures collection: Collects the user's weight and height measurements to calculate the IMC
- IMC Calculation and Display: Calculate and display the user's Body Mass Index (IMC) with an interactive graph and associated health advice.
- Personalized Product Recommendation: Suggest the CrioCaseira product based on the user's needs.
- Video Display with Delayed Button: Display a video with a call-to-action button that appears 10 seconds into the video.

## Style Guidelines:

- Primary color: Deep sky blue (#4169E1) to match the user request for an app that addresses water retention; desaturate slightly from pure blue
- Background color: Very light blue (#F0F8FF) to provide a gentle, icy backdrop, subtly reminiscent of a gel texture.
- Accent color: Teal (#008080), slightly offset to complement the primary, to highlight calls to action and interactive elements
- Headline and body font: 'Poppins' sans-serif. Note: currently only Google Fonts are supported.
- Clean and responsive design with a focus on readability, using clear sections and whitespace.
- Simple, intuitive icons for each answer option.
- Subtle loading animations and button pulsations to guide user interaction.
